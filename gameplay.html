Gameplay

Creating a Separator in MO2

  1. Right-click the empty space in the left pane of MO2 and select Create Separator.
  2. Name the separator Gameplay.
It is especially important to read the descriptions of these mods (You should still read the description of every mod), so you know what has changed and how to use the new features.

Balanced Creature Stats - Fixed Level Scaling

Installation instructions:

  • Main Files - Balanced Creature Stats
  • Optional Files - Balanced Creature Stats - DLC Patch (select the name that matches the file on Nexus when the installation prompt comes up)
Fixes Oblivion's creature level scaling by making all creature levels static or enforcing a maximum level for scaling with the player. Also balances and fixes creature stats logically across similar types, without altering the vanilla intention.

Balanced NPC Level Cap - Logical Levels for All NPCs

Installation instructions:

  • Main Files - Balanced NPC Level Cap
  • Optional Files - Balanced NPC Level Cap - DLC Patches (select the name that matches the file on Nexus when the installation prompt comes up)
The comprehensive solution to Oblivion's NPC scaling problem: adds consistent level caps to 1,400+ NPCs from the base game and all DLCs.

Balanced Unleveled Rewards

Installation instructions:

  • Main Files - Balanced Unleveled Rewards - All-in-One
Removes level scaling from all leveled quest rewards and items, instead giving them balanced stats more like how artefacts are handled in Skyrim.

Attribute Progression Redesign

Installation instructions:

  • Main Files - Attribute Progression Redesign v1.2
Unused attribute points carry over to your next level. My take on disincentivizing min-maxing approaches to gameplay while also preserving the original intended balance.
