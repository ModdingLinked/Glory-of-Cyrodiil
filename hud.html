User Interface

Creating a Separator in MO2

  1. Right-click the empty space in the left pane of MO2 and select Create Separator.
  2. Name the separator User Interface.

NorthernUI

Installation instructions:

  • Main Files - NorthernUIAway (Vanilla Style)
Patches native Xbox controller support into the game and fixes many other UI related issues.

Diverse Effect Icons

Installation instructions:

  1. Main Files - Diverse Effect Icons 1_11
  2. Once the mod has been installed, double-click on it in the left pane of MO2
  3. In the Filetree tab, right-click all ESP files and hide
Introduces unique icons for most of the magic effects which doesn't already have a unique icon.

Diverse Effect Icons OBSE

Installation instructions:

  • Main Files - DEI OBSE Patch
Diverse Effect Icons with OBSE support for better compatibility.

Make sure you followed the installation steps for Diverse Effect Icons!
